Known issues

  1. Problem: in case of Brazil timezone, during summer time shift, scheduler can render double days ( so far, there was only few reports from users of Brazil-version of Windows, problem doesn't appear for any other type of OS )
    Symptom: double days in month view
    Affected versions: all versions
    Patch: http://support.dhtmlx.com/x-files/scheduler_2_1/brazil_patch.zip
    Patch need to be included on the page after other js files.

  2. Problem: time limit set using blockTime doesn't work correctly in the Units view
    Affected versions: 3.0
    Patch: http://forum.dhtmlx.com/download/file.php?id=4139
